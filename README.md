# ReScript Country Select

## Installation

```sh
npm install
```

## Build

### 1. Build ReScript

- Build Rescript: `npm run res:build`
- Clean Rescript: `npm run res:clean`
- Build & watch Rescript: `npm run res:dev`

### 2. Build Test App

- Build Test App: `npm run parcel:build`
- Run & watch Test app: `npm parcel:start`
